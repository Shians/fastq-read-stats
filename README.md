# Fastq Read Stats

Reports a summary of the read lengths in fastq(.gz) files with the format below

```
Read length summary:

Reads processed: 71915

  Mean: 566

   Min: 55
    Q1: 399
Median: 508
    Q3: 671
   Max: 4280
```

# Installation

```
git clone https://github.com/Shians/fastq-read-stats.git
cd fastq-read-stats
cmake .
make
make install
```
